# The Lithuanian Alphabet

Lithuanian uses the Latin alphabet (like English) but adds some "diacritics" (little marks).
There are 32 letters.

## The Vowels

### Short Vowels

- **a** — like 'a' in p**a**rk.
- **e** — like 'e' in b**e**d.
- **i** — like 'i' in s**i**t.
- **u** — like 'u' in p**u**t.

### Long Vowels

- **y** — like 'ee' in sh**ee**p. (_Note: y is a vowel here!_)
- **ū** — like 'oo' in m**oo**n.
- **ą, ę, į, ų** — These are "nasal" vowels historically, but now just pronounced **LONG**.
  - **ą** = long **a** (father).
  - **ę** = long **e** (bad).
  - **į** = long **i** (sheep - same as **y**).
  - **ų** = long **u** (moon - same as **ū**).

### The Special Dot: Ė

- **ė** — like the 'ai' in **ai**r (narrow, no diphthong). Start saying "Eh" and smile.

## The Consonants

Most are like English, with key exceptions:

- **j** — like 'y' in **y**es. (Jonas = Yonas).
- **c** — like 'ts' in ca**ts**. (Centas).
- **č** — like 'ch' in **ch**ips. (Čia = Here).
- **š** — like 'sh' in **sh**ip. (Šuo = Dog).
- **ž** — like 's' in plea**s**ure. (Žmonės = People).

## Mini FAQ

### Why are there two "i"s? (i and y)

- **i** is short.
- **y** is long.
- Meaning changes! _Pilis_ (Castle) vs _Pylis_ (Pouring).

### What is the "h" sound?

Lithuanian "H" is soft. "Ch" (choras) is harder, like Loch Ness.
